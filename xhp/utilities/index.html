Utilities

XCC Game Spy

This utility can record Red Alert 2 multiplayer games which can be replayed with the included player. XCC Utilities
This archive contains XCC AV Player, Editor, Mixer, MIX Editor, Mod Creator, Launcher and TMP Editor.

XCC AV Player

This is a program that allows you to play the audio & video files of C&C:Tiberium Dawn and C&C:Red Alert directly from the MIX files.
The videos are played including sound. The videos are displayed in 320x200x32. If that resolution/color depth is not supported, 16 bit color or 640x480 will be used.
It's now also possible to play and extract the audio of C&C:Tiberian Sun.

XCC Editor

A map editor like CC Map for C&C: Tiberian Dawn.
Make sure you've got conquer.mix, desert.mix, temperat.mix and winter.mix in your search path.

XCC Mixer, with page

A mix manager like Mix Manager for C&C: Tiberium Dawn and RAMIX for C&: Red Alert. But this one is for C&C: Tiberian Sun, C&C: Red Alert 2, C&C: Renegade and C&C: Generals, while you can also use it for Dune 2, C&C: Tiberian Dawn and C&C: Red Alert. It contains viewers for CPS, DDS, PCX, SHP, TXT and more. You can extract everything.

XCC Mod Creator/Launcher, with page

This utility allows you to create and launch Mods for C&C: Tiberian Sun and C&C: Red Alert 2.

XCC Map Encoder/Decoder, with page

This utility encodes TS/RA2 maps to a format that's 3x smaller then ZIP.

XCC Radar Customizer

This utility changes the brightness/color of the terrain on the radar of Tiberian Sun and Red Alert 2.

XCC RA2 Map Updater, with page

This utility retrieves all map packs released by Westwood Studios before it launches Red Alert 2.

WOL Matching Filter (MF), XCC WOL IRC Filter (XWIF), with page

This utility can be used as a bot on WOL for quick and custom match. It notifies you of the rank of your opponent. It also kicks opponents that are filtered out.

XCC Sound Editor, with page

This utility allows you to insert, play, extract and delete sounds of Red Alert 2. XCC Sound Editor is part of XCC Mixer.

XCC String Table Editor, with page

This utility allows you to insert, edit and delete strings of Red Alert 2. XCC String Table Editor is part of XCC Mixer.

XCC MIX Editor

This utility allows you to create and/or edit MIX files.

XCC INI CLT, with page

This is a command line tool that allows you to add or substract two INI files.

XCC MIX CLT

This is a command line tool that allows you to extract files from a MIX file.

XCC AUD Writer Plug-In, with readme

This is a plug-in for WinAmp that allows you to write AUD files, which can be used for C&C: Tiberium Dawn, C&C: Red Alert and C&C: Tiberian Sun and other games.

XCC IMA ADPCM WAV Writer Plug-In, with readme

This is a plug-in for WinAmp that allows you to write IMA ADPCM WAV files, which can be used for C&C: Red Alert 2 and other games.
